# E2E Plan: login  ✅ 承認済み（承認ゲート①通過・Step3 codegen 入力）

> 生成日: 2026-06-25 / 入力源: 実サイト探索(chrome-devtools) + seed test
> 対象URL: http://localhost:3000/login
> 認証: 要 / 方式=form(email+password) / ロール=未ログイン・user・admin
> 探索環境: ロケール=ja-JP / タイムゾーン=Asia/Tokyo

## 遷移マップ（要約）
- `/login` フォーム認証。成功で `/dashboard`、失敗(401)で `/login` にエラー表示、空 submit で検証エラー。
- 認証戦略: storageState 前提。setup が user でログインし `e2e/.auth/user.json` を作る。各テストは state で開始。
- admin は `e2e/.auth/admin.json` を別途用意し管理メニューの対を検証。

## 破壊的シナリオの決定（一括提示の結果）
破壊的シナリオなし（本 feature はログインフロー＋閲覧のみ。teardown 不要）。

## シナリオ仕様

### S1. ログイン成功（happy path）
- **開始状態**: 未ログイン / 有効ユーザー存在 / `/login`
- **操作**: 1. メール `user@example.com` 入力 2. パスワード `password` 入力 3. 「ログイン」押下
- **中間観測点**: 押下後ローディング表示 → `POST /api/login` 完了で `/dashboard` 遷移
- **終了条件**: ダッシュボード見出しと「user@example.com」表示
- **後始末**: -
- **除外事項**: パスワードリセット導線

### S2. 必須項目未入力（validation error）
- **開始状態**: 未ログイン / `/login`
- **操作**: 空のまま「ログイン」押下
- **中間観測点**: `/api/login` は呼ばれない
- **終了条件**: 「メールアドレスを入力してください」表示
- **後始末**: - / **除外事項**: -

### S3. 不正な認証（認証失敗 401）
- **開始状態**: 未ログイン / `/login`
- **操作**: 不正なパスワードで submit
- **中間観測点**: `POST /api/login` が 401
- **終了条件**: 「メールアドレスまたはパスワードが違います」表示、`/login` に留まる
- **後始末**: - / **除外事項**: アカウントロック回数制御

### S4. 戻る操作
- **開始状態**: storageState=user で `/dashboard`
- **操作**: ブラウザの戻る
- **終了条件**: 未ログインに戻らず `/dashboard` 維持（または `/login` へ正しくリダイレクト）
- **後始末**: - / **除外事項**: -

### S5. 再読込
- **開始状態**: storageState=user で `/dashboard`
- **操作**: リロード
- **中間観測点**: セッション復元
- **終了条件**: ログイン保持で `/dashboard` 表示
- **後始末**: - / **除外事項**: -

### S6. 途中離脱
- **開始状態**: `/login` でメールのみ入力
- **操作**: 別ページへ遷移し `/login` に戻る
- **終了条件**: 入力途中値の保持仕様に従う（保持しないなら空）
- **後始末**: - / **除外事項**: -

### S7. ネットワーク遅延（happy path の遅延版）
- **開始状態**: 未ログイン / `/login` / `/api/login` を 3s 遅延にモック
- **操作**: 有効な認証で submit
- **中間観測点**: 応答までローディング表示が継続、ボタン二重押下不可
- **終了条件**: 応答後 `/dashboard` 遷移
- **後始末**: - / **除外事項**: タイムアウト時のリトライUI

### S8. 権限差分（permission / user に管理メニュー非表示）
- **開始状態**: storageState=user で `/dashboard`
- **操作**: ナビゲーションを確認
- **中間観測点**: 「user@example.com」表示
- **終了条件**: 「管理メニュー」リンクが**非表示**（admin 側は storageState=admin で「表示」の対を検証）
- **後始末**: - / **除外事項**: 管理機能の各操作の中身
